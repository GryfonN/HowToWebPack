# HowToWebPack

Test projekt, pre vyskúšanie si práce s Webackom. **Pozrieť si commit log a diff-y s #HOWTO tagom.**

## Požiadavky
- NodeJS v5.2.0
- Npm 3.7.3
    - gulp-cli 1.2.1 


## Ako spusťiť
- *npm install*
    - počas inštalácie sa spustí semantic-ui wizard, ten stačí 3x potvrdiť
- *cd ./semantic/* a spustiť build *gulp build*
- *npm run start*
